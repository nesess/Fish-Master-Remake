# Fish-Master-Remake
A remake for popular mobile game "The Fish Master!".
I do not have any money making purpose. Made for educational purposes only.
Orginal game: https://play.google.com/store/apps/details?id=io.voodoo.fishmaster&hl=tr&gl=US

I do not have any money making purpose. Made for educational purposes only.

Info:

The goal in this game is fish different types of fishes and earn money. With this money you can upgrade your power (how many fishes can you catch), rope length (to reach deeper and catch more valuable fishes) and more offline money (while you not playing you can earn money).  I can share game files if requested. I also add show ads system with this you can earn 2 times more money. I did this to learn how to implement ads system.
